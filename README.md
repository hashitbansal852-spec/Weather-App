# 🌦️ Weather Dashboard

A modern weather dashboard built using **HTML**, **CSS**, and **JavaScript** that provides real-time weather information using the **WeatherAPI**.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature in Celsius
- 💧 Humidity information
- ☁️ Current weather condition
- 🌤️ Dynamic weather icons from the API
- ❌ Invalid city handling
- 🎨 Modern glassmorphism UI
- 📱 Responsive and clean design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI

## ⚙️ How It Works

1. Enter a city name.
2. Click the **Search** button.
3. The application fetches live weather data from WeatherAPI.
4. Weather information is displayed instantly.
5. Invalid inputs display an appropriate error state.
